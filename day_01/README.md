# Day 1

## Requirement

* [Julia](https://julialang.org/) (confirmed to work with Julia v1.11.1)

## How to use

### Read the source file in REPL

```console
julia> include("d01.jl")
```

### Run each solver with the name of the puzzle data file as an argument

*Part one*

```console
julia> d01_p1("input")
```

*Part two*

```console
julia> d01_p2("input")
```
