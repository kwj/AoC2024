# Day 4

## Requirement

* [Julia](https://julialang.org/) (confirmed to work with Julia v1.11.2)

## How to use

### Read the source file in REPL

```console
julia> include("d04.jl")
```

### Run each solver with the name of the puzzle data file as an argument

*Part one*

```console
julia> d04_p1("input")
```

*Part two*

```console
julia> d04_p2("input")
```
